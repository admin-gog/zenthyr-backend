## Running Locally

1. Clone the repository:

```bash
git clone https://github.com/admin-gog/zenthyr-backend.git
```

2. Navigate to the project directory:

```bash
cd zenthyr-backend
```

3. Install dependencies:

```bash
npm install
```

4. Create a .env file:

   - Create  `.env` similar to `.env.example`.

5. Start the development server:

```bash
npm run start
```
